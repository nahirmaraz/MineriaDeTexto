  # Minería De Texto - PLN
  ## Objetivo
  Generar una nube de palabras (*WordCloud*) donde el tamaño de cada palabra representa la frecuencia con que aparece en el texto.
  ## Introducción
  Este proyecto aplica técnicas de Procesamiento de Lenguaje Natural (PLN) para analizar la letra de una canción.
  El proceso consiste en:

1. Carga de la letra de la canción.
2. Tokenización del texto.
3. Conversión a minúsculas.
4. Eliminación de signos de puntuación y *stop words*.
5. Cálculo de la frecuencia de cada palabra.
6. Visualización mediante una nube de palabras.
 ## Tecnologías utilizadas

- Python: Pandas, NumPy, NLTK, WordCloud, Matplotlib.
## Habilidades demostradas

- Preprocesamiento y limpieza de texto.
- Manipulación de datos con Python.
- Uso de bibliotecas para análisis de datos (Pandas, NumPy).
- Aplicación de NLTK para tokenización y eliminación de *stop words*.
- Visualización de resultados mediante WordCloud y Matplotlib.
